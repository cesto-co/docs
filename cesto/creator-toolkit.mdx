## What is the Creator Toolkit Skill?

The Creator Toolkit Skill is an open-source AI agent skill for **creators**. Where the [Cesto Toolkit Skill](/cesto/skill) is for browsing, analyzing, and posting to Labs, the Creator Toolkit is for building the real thing: investable product baskets, with versioning, metadata, and AI-generated cover art, all through natural conversation.

It works with any AI coding agent that supports skills: Claude Code, Cursor, Windsurf, and more.

<Note>
This skill requires **creator access**. It checks your role on first use and stops if your account isn't a creator. See the [Creator program](/cesto/creators) to get started.
</Note>

## Installation

The skill lives at [`cesto-co/cesto-skills`](https://github.com/cesto-co/cesto-skills) on GitHub.

<CodeGroup>
```json Claude Code (.claude/settings.json)

  "skills": [
    "github:cesto-co/cesto-skills/cesto-creator-toolkit"
  ]
}
```

```json Cursor (.cursor/skills.json)
{
  "skills": [
    "github:cesto-co/cesto-skills/cesto-creator-toolkit"
  ]
}
```


## Authentication

Like the Cesto Toolkit, it uses a magic-link wallet login: connect your Solana wallet in the browser and sign once. Credentials are stored at `~/.cesto/auth.json` and refresh silently. Before doing anything, the skill verifies your account has a **creator** (or admin) role.

## What you can do

### Create a basket

```
Create a basket called "AI Infrastructure" with 40% RENDER, 35% TAO, 25% GRT
```

You'll review the allocation, metadata, and cover before it's saved as a draft.

### Edit a basket

```
Update the description and risk notes on my "AI Infrastructure" basket
```

### Rebalance (new version)

```
Rebalance "AI Infrastructure" to 50% TAO, 30% RENDER, 20% GRT
```

### Generate cover art

```
Generate a cover image for my AI Infrastructure basket
```

## API endpoints

| Endpoint | Method | Description |
|---|---|---|
| `/users/me` | GET | Confirm your account and role |
| `/tokens` | GET | Supported tokens with prices |
| `/products?mine=true` | GET | Your own baskets (live + draft) |
| `/creator/products` | POST | Create a basket (draft) |
| `/creator/products/{id}` | PUT | Update basket metadata |
| `/creator/products/{id}/versions` | POST | Publish a new version (rebalance) |
| `/creator/products/versions/{id}` | PUT | Patch version metadata (changelog, minimum investment, …) |
| `/positions/simulate` | POST | Backtest a basket's allocation |

The skill also provides cover-image endpoints for uploading your own art or generating it with AI.

<Note>
New baskets and new versions are always created as **drafts**. Publishing to the public catalog is handled by the Cesto team.
</Note>
